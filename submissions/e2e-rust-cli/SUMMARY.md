# e2e-rust-cli
A Rust CLI tool that provides echo functionality and cryptocurrency price queries through onchainos integration for end-to-end testing.

## Highlights
- Echo command for basic argument testing
- ETH price queries via integrated CLI
- BTC/WBTC price queries through onchainos
- Built-in onchainos integration for market data
- Comprehensive error handling for common issues
- TLS/certificate error workarounds
- Pre-flight checks for required dependencies
- JSON output format for price data
